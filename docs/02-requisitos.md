# Requisitos do Sistema

## Requisitos Funcionais

### RF001 - Cadastro de Clientes

O sistema deve permitir cadastrar, editar, consultar e excluir clientes.

### RF002 - Cadastro de Veículos

O sistema deve permitir cadastrar veículos vinculados aos clientes.

### RF003 - Ordem de Serviço

O sistema deve permitir abrir, editar e finalizar Ordens de Serviço.

### RF004 - Controle de Peças

O sistema deve permitir cadastrar peças e controlar estoque.

### RF005 - Movimentação de Estoque

O sistema deve registrar entradas e saídas de produtos.

### RF006 - Financeiro

O sistema deve controlar contas a receber e contas a pagar.

### RF007 - Relatórios

O sistema deve gerar relatórios operacionais e financeiros.

### RF008 - Dashboard

O sistema deve apresentar indicadores da oficina.

### RF009 - Usuários

O sistema deve permitir autenticação e controle de acesso.

---

## Requisitos Não Funcionais

* Desenvolvido em ASP.NET Core MVC
* Banco PostgreSQL
* Arquitetura em Camadas
* Entity Framework Core
* Compatível com Windows e Linux
* Controle de versão com GitHub
* Interface responsiva
* Segurança baseada em ASP.NET Identity
