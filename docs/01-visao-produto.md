# Visão do Produto

O Veltor ERP é um sistema de gestão empresarial focado inicialmente em oficinas mecânicas.

O sistema permitirá controlar:

* Clientes
* Veículos
* Ordens de Serviço
* Estoque de peças
* Compras
* Financeiro
* Relatórios
* Usuários e permissões

O projeto será desenvolvido utilizando ASP.NET Core MVC, PostgreSQL e arquitetura em camadas, permitindo evolução futura para modelo SaaS.
