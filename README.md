# Veltor ERP

## Controle Total da Sua Oficina

Veltor ERP é um sistema de gestão para oficinas mecânicas desenvolvido com ASP.NET Core MVC, Entity Framework Core e PostgreSQL.

O objetivo é fornecer uma plataforma moderna para gerenciamento de clientes, veículos, ordens de serviço, estoque, financeiro e indicadores de desempenho.

## Tecnologias

* ASP.NET Core MVC
* Entity Framework Core
* PostgreSQL
* Bootstrap
* GitHub
* Arquitetura em Camadas
