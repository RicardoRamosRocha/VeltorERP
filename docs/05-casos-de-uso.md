# Casos de Uso

## UC001 - Cadastrar Cliente

Ator: Atendente

Fluxo:

1. Acessar menu Clientes
2. Clicar em Novo Cliente
3. Informar dados
4. Salvar

Resultado:
Cliente cadastrado com sucesso.

---

## UC002 - Cadastrar Veículo

Ator: Atendente

Fluxo:

1. Selecionar cliente
2. Informar dados do veículo
3. Salvar

Resultado:
Veículo vinculado ao cliente.

---

## UC003 - Abrir Ordem de Serviço

Ator: Mecânico

Fluxo:

1. Selecionar cliente
2. Selecionar veículo
3. Informar problema
4. Salvar

Resultado:
OS aberta.

---

## UC004 - Finalizar Ordem de Serviço

Ator: Mecânico

Fluxo:

1. Informar serviços executados
2. Informar peças utilizadas
3. Encerrar OS

Resultado:
OS finalizada.

---

## UC005 - Registrar Entrada de Estoque

Ator: Estoquista

Fluxo:

1. Selecionar produto
2. Informar quantidade
3. Confirmar

Resultado:
Estoque atualizado.
